Esame di Tecniche di Programmazione del 27/05/2015

Istruzioni per lo svolgimento della prova:

  • Inserire nel file studente.txt i dati personali richiesti (matricola, nome e cognome) senza toccare altre parti del file, quindi salvare e chiudere il file.
  • All’esercizio i è associata la directory esercizio{i}, contenente i file necessari allo svolgimento dell’esercizio. Ciascun esercizio deve essere svolto modificando esclusivamente il file esercizio{i}.h (se necessario) ed esercizio{i}.c presenti nella directory corrispondente. Nessun file per nessun motivo deve essere spostato dalla directory in cui si trova. Nessun altro file deve essere modificato. Non devono essere creati ulteriori file.
  • Il file contenente il programma principale di ciascun esercizio è test_esercizio{i}.c. Tale file deve essere compilato usando il Makefile fornito nella directory, ed il corrispondente eseguibile (test_esercizio{i}) lanciato per eseguire i test di correttezza e conoscerne l’esito. Il file test_esercizio{i}.c non deve essere modificato.
  • Alla fine del tempo concesso per svolgere l'esame o, se avete terminato l'esame prima dello scadere del tempo, dovete eseguire lo script di consegna dell'esame. La consegna può essere effettuata una ed una volta sola, per cui consegnate solo quando siete sicuri di volerlo fare. Lo script è fornito insieme alla documentazione dell' esame e per eseguirlo basterà andare tramite terminale nella cartella dove si trova lo script consegna_esame.sh e avviarlo da linea di comando. Il comando da lanciare risulterà simile al seguente: "./consegna_esame.sh". Lo script di consegna deve essere eseguito dalla directory in cui esso si trova, in caso contrario farà errore. Una volta avviato lo script controllate che la consegna sia andata a buon fine verificando che sul terminale venga stampato un output simile al seguente:

    local: matricola_cognome_nome_hostname.tar.gz remote: uploads/matricola_cognome_nome_hostname.tar.gz

    200 PORT command successful. Consider using PASV.

    150 Ok to send data.

    226 Transfer complete.

    4886 bytes sent in 0.00 secs (113606.8 kB/s)

    221 Goodbye.

  • Verranno corretti esclusivamente gli esercizi la cui compilazione dia esito positivo. In caso contrario veràr assegnato un punteggio nullo. Se non verrà eseguito lo script di consegna dell'esame esso non verrà corretto (è equivalente a a non aver consegnato il compito a fine esame).
  • Per eventuali problemi o dubbi, chiedere al docente.

Esercizio 1

Implementare la funzione C:

  • int delete_occurrences(TipoSCL* scl, int info);

che dato un puntatore ad una lista collegat scl, e un intero info, elimini tutte le occorrenze di elementi nella lista scl che hanno il campo informazione uguale ad info. Inoltre come valore di ritorno deve restituire un intero contenente il numero di elementi eliminati dalla lista collegata scl.

Esempio

Dato la seguente lista:

scl = [2 -> 5 -> 3 -> 2 -> 8 -> 2 -> 2 -> 9 -> 4 -> 2]

e richiamando la seguente linea di codice:

delete_occurrences(scl, 2);

scl dovrà contenere:

scl = [5 -> 3 -> 8 -> 9 -> 4]

e la funzione dovrà restituire un intero di valore 5.
